# Personal Website Application

## Requirements

- Python 3.14.5
- `pip`
- Optional: a virtual environment tool such as `venv`

## Setup

### 1. Clone the repository

Using SSH:

```bash
git clone git@github.com:cmlocric/jhu_software_concepts.git
cd jhu_software_concepts
```

### 2. Create and activate a virtual environment

#### macOS / Linux

```bash
python -m venv .venv
source .venv/bin/activate
```

#### Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```
