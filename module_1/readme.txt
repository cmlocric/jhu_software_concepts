Instructions for running personal website application

Requirements
•	Python 3.14.5
•	pip
•	Optional: virtual environment tool such as venv

Setup
1. Clone the repository
Using SSH:
git clone git@github.com:cmlocric/jhu_software_concepts.git
cd REPO

2. Create and activate a virtual environment
macOS/Linux:
python -m venv .venv
source .venv/bin/activate
Windows:
python -m venv .venv
.venv\Scripts\activate

3. Install dependencies
pip install -r requirements.txt
